This project focuses on predicting the impact of electronic gadget addiction on students' lives using machine learning. By collecting data on students' gadget usage habits and their effects on academic performance, social interactions, and mental health, the project aims to identify different stages of addiction. Machine learning models will be employed to analyze this data, providing insights and tailored recommendations to help students manage and reduce their dependency on electronic devices, ultimately promoting a healthier balance between technology use and daily life.
